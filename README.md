# workflow-backend
backend for workflow management system



**Allowed Inputs**

1. User Designations
    - HOD
    - SENIOR_TREASURER
    - JUNIOR_TREASURER
    - PRESIDENT
    - SECRETARY
    - COORDINATOR
  
2. Event Inspectors Status
    - APPROVED
    - REJECTED
    - PENDING
  
3. Event Status
    - DONE
    - PUBLISHED
    - PENDING
    - CONFIRMED
    - REJECTED
 
4. User Status
    - ACTIVE
    - INACTIVE
